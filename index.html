обучение гита

  • git init - это создать хранилище (репозиторий)
  • git confic user email.это -для связи с тем кто писал этот код
  • git confic user email.name свое имя -имя человека кто это делал
  • git add - добовляет файл область сохранения
  • git status -статус области сохранения
  • git commit -m-создание точки сохранения
  • git config gui.encoding utf-8 -для корректного отображения языка
  • git-remote add -для названия удаленного доступа. и ссылка удаленного доступа
  • git push- название удаленного доступа и ветки -- и сайт зальется в интернет
  • sshkeygen - создание безопасного ключа для подключения к репозиторий
